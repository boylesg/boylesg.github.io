Purpose:

To implement the functionality of an array buffer. Unlike a native C array, CBuff, CFixedBuff and CDynamicBuff intialise their internal arrays to 0's on declaration.
            CBuff is a template class where you declare its maximum size like this: CBuff<100>; buff.  It avoids the use of dynamic memory, which can cause memory
            fragmentation in the limited memory of an Arduino MCU.CFixedBuff allows you to use any externally declared array as the internal buffer for CFixedBuff.

            CDymanicBuff uses dynamic memory and lets you declare a  CDynamicBuff object like this: CDynamicBuff buff(100).
            It should be used sparingly with Arduino MCUs. 

            All 3 classes are derived from CBuffBase and objects of each class can be used interchangeably as pointers:

            CBuff<100> buff1;
            CDynamicBuff buff2(100);
            CBuffBase pBuff = &amp;buff1;
            pBuff = &amp;buff2;

Dependancies:

None

Constructors:
  

CBuff<SIZE> buff()

SIZE: template parameter that specifies the size of the internal array.

CFixedBuff(const char *strBuff, const uint16_t nBuffSize, const bool bClear)

strBuff: a pointer to a byte array. It can be a char,

 uint8_t or int8_t array.

nBuffSize:the capacity of the buffer strBuff.

bClear: true if you want the array to be initialised

to 0's, false if not.

CDynamicBuff(const uint16_t nBuffSize)

nBuffSize: the maximum capacity of the buffer.
 

Functions:

void empty()

                        Sets all elements to 0.

int capacity()

                        Returns: the maximum capacity of the buffer.

int length()

                        Returns: assuming the buffer contains a string then it return the length of the string, up to the first NULL terminator.

void setAt(const uint16_t nI, const char cCh)

                        Sets to value at the specified index to a newvalue.
                        nI: the index of the value to be changed.
                        cCh:the new value.

            char getAt(const uint16_t nI)

                        Gets the value at the specified index.
                        nI:the index of the value to be fetched.
                        Returns: the value at the specified index.

operator char*()

                        Converts the buffer object to a native C string.

bool enqueue(const char cCh)

                        Allows you to treat the buffer as a queue or FIFO data structure. It adds the specified new value to the end of the internal array.
                        Returns: true if the length of the buffer is less than it maximum capacity, otherwise false.

char dequeue()

            Allows you to treat the buffer as a queue or FIFO data structure. It removes the value at the beginning of the array and moves all values after it one
            place toward the beginning of the array.
            Returns: the value removed if the array is not empty, otherwise 0.

bool push(const char cCh)
                        Allows you to treat the buffer as a stack or LIFO data structure. It adds the specified new value to the end of the internal array.
                        cCh: the new value to be added.

char pop()
            Allows you to treat the buffer as a stack or LIFO data structure. It removes the value at the end of the internal array and returns it.
            Returns: the value that was removed if the internal array is not empty, otherwise 0.
