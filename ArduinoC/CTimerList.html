Purpose:

             To implement a list of polled tasks as a function to execute per a time interval in seconds.

 

Dependencies:

Constants.h, Common.h/cpp, CBuff.h/cpp, CString.h/cpp, debug.h/cpp

 

class CTimerList

                Constructors

                                CTimerList()

                                                Creates an empty list.

 

                Functions

                                void poll()

This function must be called in loop() in order for the timed functions to be executed.

                                void clear(const uint8_t nI)

                                                Erases the specified timed function without removing it.

                                                nI: index of the timed function to be cleared.

                                void remove(const uint8_t nID)

                                                Removes the specified timed function from the list.

nID: ID of the timed function to removed.

                                CTimer* add(const uint8_t nOwnerID, const uint32_t nIntervalSec, const GlobalFunctionPtrType pCallbackFunc)

                                                Adds a timed function to the list.

nOwnerID: a unique integer ID for the function.

nIntervalSec: the time interval that the function is to be called at.

pCallbackFunc: a pointer to the function to be called.

                                uint8_t getSize()

                                                Returns the current number of timed functions in the list.

                                int8_t getLast()

                                                Returns the index of the last timed function.

                                void dump()

                                                For debugging purposes - outputs the contents of the list to Serial Monitor.

Example Usage

CtimerList TL;

 

void setup()

{

TL.add(10, 20, Function1);

TL.add(20, 1, Function2);

TL.add(30, 5, Function3);

TL.clear(TL.GetLast());

TL.remove(10);

TL.dump();

}

 

void loop()

{

TL.poll();

}
