Purpose: To implement the functions of an array without having to worry about array index over run errors. It is a template class where you specify the data type it will contain and the maximum size. The class does not use dynamic memory (new, alloc and malloc), which can cause memory fragmentation with the limited memory available on Arduino microcontrollers. The first template parameter (int in the above example) specifies the data type to be stored and the second template parameter (100 in the above example) specifies the size of a regular C array inside the class.

Dependancies: None

Queue Functions:

These functions allow you to treat the array as a queue or FIFO data structure.

bool enqueue(DataType DataVal)

Add a new item at the end or array. The array length will increase by 1.

                                DataVal: New value to add.        

Returns: True if the number of items in the array is less than the maximum length, otherwise false.

bool dequeue(DataType &DataVal)       

Remove an item from the beginning of the array. All other items in the array will be moved toward the beginning of the array by one place. The array length will decrease by 1.

                                DataVal: The value removed is returned in this parameter.

Returns:  True if the array is not already empty, otherwise false.

Stack Functions:
                These functions allow you to treat the array as a stack or LIFO data structure.

                bool push(DataType DataVal)

                                Add a new item at the end of the array. The array length will increase by 1.

DataVal: The value to be pushed.

Returns: True if the array is not already at its maximum length, otherwise false.

bool pop(DataType &ampDataVal)

Remove an item from the end of the array. The array length will decrease by 1.

                                DataVal: The value removed is returned in this parameter.                                                                          

Returns: True if the array is not already empty, otherwise false.

List functions:

                 These functions allow you to treat the array as a list data structure.

bool add(DataType DataVal)

Add a new value at the end of the array. The array length will increase by 1.

DataVal: The value to be added.

                                Returns: True if the array is not already at its maximum length, otherwise false.

bool insertAt(const uint8_t nI, DataType DataVal)

Insert a new value into the array after the specified index. The item at that index and all other after it will move towards the end of the array by one place. The array length will increase by 1.

nI: The index the new value will be inserted at.

                                DataVal: The value to be inserted.

Returns: True if the array is not already at its maximum length, otherwise false.

bool removeAt(const uint8_t nI, DataType &ampDataVal)

Remove an item from the array at the specified index. All other items after that index will move towards the front of the array by one place. The array length will decrease by 1.

nI: The index of the value that will be removed.

                                DataVal: The removed value is returned in this parameter.

                                Returns: True if the array is not already empty, otherwise false.

                                                                               

Array functions:

                These functions are the regular array indexing functions.

                bool setAt(const uint8_t nI, DataType DataVal)

Change the value of the item at the specified indexto the new specified value.

                                nI: The index of the value that will be changed.

                                DataVal: The new value.

                                Returns: True if the nI is in range, otherwise false.

                bool getAt(const uint8_t nI, DataType &ampDataVal)

                                Return the value at the index specified by parameter nI.

                                nI: The index of the value that will be fetched.

                                DataVal: The value is returned in this parameter.

                                Returns: True if the nI is in range, otherwise false.

                DataType operator [](const uint8_t nI)

                                Returns the value at the index specified by parameter nI.

                                nI: The index of the value that will be fetched.

                                Returns: The value at the specified index.

General functions:

                These functions functions.                          

                 uint8_t size()

                                Returns: the current size of the array, not the maximum size.

                 uint8_t capacity()

                                Returns: the maximum size of the array.

                bool isEmpty()

                                Returns: true if current size of the array is 0, otherwise false.

                void empty()

                                Empties the array of all values so that its size becomes 0.

                bool find(DataType DataVal)

                                Searches for a value in the array.

                                DataVal: The value to search for.

                                Returns: True if the value is found, otherwise false.
