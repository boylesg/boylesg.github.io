Purpose:

To implement date and time handling in self contained objects.

 

Dependancies:

Arduino RealTimeClockDS1307.h, Constants.h, CBuff.h/cpp, CString.h/cpp, Common.h/cpp.

 

Functions:

bool isLeapYear(const uint16_t nYear)

Returns true if the year specified in the parameter is a leap year, otherwise false.

 

uint8_t getMaxDOM(const uint8_t nMonth, const uint16_t nYear)

Returns the maximum day number for the month specified in the parameter.

 

void getDayName(const uint8_t , const bool bShort, CString &strDayName)

Returns the name of the day, e.g. Monday, in the parameter strDayName. The parameter nDayNum specifies the number of the day within the week (1 - 8).
The parameter bShort specifies whether the day name is in long or short format e.g. Monday or Mon.

 

void getMonthName(const uint8_t nMonthNum, const bool bShort, CString &strMonthName)

Returns the name of the month, e.g. January, in the parameter strMonthName. The parameter nMonthNumspecifies the number of the month within the
year (1 - 12). The parameter bShort specifies whether the month name is in long or short format e.g. January or Jan.

 

uint8_t calcDOW(const uint8_t nDOM, const uint8_t nMonth, uint16_t nYear)

Calculates the day of week given the day number in the month, in parameter nDOM, the month number and the year.
