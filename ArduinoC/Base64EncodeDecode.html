Purpose:

To implement HTML encoding and decoding. Non alphanumeric characters in HTML requests are encoded as hex values, e.g. %50

                                                                               

Dependancies:

CBuff.h/cpp, CString.h/cpp, Common.h/cpp, Constants.h & CMap.h.

 

Functions:

                void encodeBase64(CString & strDataIn)

                                Base64 encodes the string specified in parameter strDataIn.

 

                void decodeBase64(CString & strDataIn)

                                Base64 decodes the string specified in parameter strDataIn.
