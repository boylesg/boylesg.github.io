What is python?

It is an interpretted programming language. This means that the software you code with Python is distributed as text based instructions that are compatible with any operating system and computer hardware. When you run the Python software an interpretter running on the coomputer reads the text based instructions and converts them to binary instructions that can be executed by the specific type of computer and operating system that the interpretter is running on.

Advantages

  • The code is plaform (operating system and hardware) independant.
  • Text based files are smaller and easier to distribute than compiled binary files.
  • They are more flexible in that they allow dynamic data typing.

Disadvantages

  • Interpretted programs run slower and hence they can not be used in time critical appications like avionics (flight control).
  • It cannot be used to code many microcontrollers due to the fact that they do not have enough memory to run large and complicated interpretter programs.
  • Interpretted languages are less secure than compiled languages.

Is Python easier to learn?

The short answer is no, it is no easier to learn than other languages. Beginners get just as confused with Python's tab system as they do with C's semi colons and braces. And learning the syntax of a programming language is a very minor part of learning to code. Learning how to use variables, data structures, loops and conditional statements, and to build software with them is hard. And how easy you find the syntax of the language is largely irrelevant.
