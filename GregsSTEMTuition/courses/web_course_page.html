Introduction

In this section I will teach you to build a basic web page using a variety of common constructs, such as popup menus and navigation structures etc. I will do this in a series of screen capture videos but I will provide minimal ongoing commentary as I do it. Just watch as I edit the code and view the results in the web browser and should should get the idea.

It would be best if you open MS Expression Web and do the same file editing as I am doing in my source file. You can pause the video as you repeat each of the editing steps that I do. By the end of it you should reasonably confident about the coding techniques.

Page structure

To begin with we are going to create the following page structure. It is just one example of how you can structure your web pages and they way you can do so is limited only by your imagination. There is no right or wrong way to structure them.

You will quickly get to know the CSS attributes I am using in the video, as these ones are very frequently used. You can jump back to my CSS page to refresh your memory as to the values available for those CSS attributes. Just follow along in MS Expression Web as I play around with different CSS attributes and values to make the page appear like the in the image above.



In fact why don't you play around with the different values for 'float' and 'display' etc., and see what effect they have on the layout for your page. Try some different values for the 'height' and 'width' atttributes. Try some different back ground colours. This is by far the best way to memorise them and to get to understand what they do. You can do in the following ways:

  • Use comments in you CSS code:

    .navigation
    {
        background-color:green;
        display:inline-block;

    .navigation
    {
        background-color:green;
        display:inline-block;
        /*float:left;*/

        width:15%;

  • Just override CSS attributes:

    .navigation
    {
        background-color:green;
        display:inline-block;
        float:left;

        float:right;
    }

    In this case the last value ('right') set for 'float' overrides the first value ('left').

Company logo and name

In this next video we will add a company logo image and company name to the banner section of the web page. We will get them to sit side by side and align at the vertical center of the banner. First we will add and <img> and a <h1> tag inside the 'banner' element like this:

<div class="banner" id="banner">
    <img id="logo" class="logo" src="Logo.jpg" alt="Logo.jpg" />
    <h1 id="title" class="title">COMPANY NAME</h1>
</div>

Now we need to play around with the CSS attributes of these two new elements to get them nicely spaced and positioned along the center line of the banner, and adjust the size of the font for the web site title. The easiest way to vertically center elements is to use the 'position' attribute with 'relative' and the 'top' attribute with a negative value to make it move up or a positive value to move it down. For both elements we use the 'display' attribute with 'inline-block' to get them to line up side by side.

.logo
{
    width:110px;
    display:inline-block;
    margin-left:1em;
    position: relative;
    top:7.7px;
}

.title
{
    display:inline-block;
    font-size:5em;
    background-color:aqua;
    margin-left:0.1em;
    position:relative;
    top:-22px;
}

By setting 'position' to 'relative', any value you then apply to the 'top' attribute moves the element up or down by that length rather than moving the top edge of the element literally to that position. A negative value means that the element will be moved up by that length while a positive value means that the element will be moved down by that length. This is what I mean when I say you just have to experiment with different combinations of CSS attributes and values until you get the elements to behave as you want them to.

Navigation structure

You can use the <nav> tag, but the <ul> is also often used to create a navigation structure. There are many ways you can implement it:

  • Oriented horizontally across the top of the page, possibly with popup sub-menus appearing below them.

  • Oriented vertically down the left side of the page, possibly with popup sub-menus appearing beside them.

  • With text hyperlinks.

  • With clickable buttons.

  • With clickable images.
The possibilities are only limited by your imagination, and perhaps how many pages there are in your web site. We will build a simple vertical text hyperlink based navigation structure in the 'navigation' element of our web page. So first we will add the following code within our 'navigation' element. The only other way you can do it is with Javascript. So what we will do is create a new file called 'common.js'. And in this file we will code a javascript function that writes the HTML code for our navigation structure.

function CreateNavigationStructure()
{
    document.write("<nav id=\"nav\" class=\"nav\">");
    document.write("<a class=\"nav_link\" href=\"index.html\">Home</a>");
    document.write("<br/>");
    document.write("<a class=\"nav_link\" href=\"products.html\">Products</a>");
    document.write("<br/>");
    document.write("<a class=\"nav_link\" href=\"services.html\">Services</a>");
    document.write("<br/>");
    document.write("<a class=\"nav_link\" href=\"contact.html\">Contact</a>");
    document.write("</nav>");
}

So what we have done is copied the all the HTML tags of our navigation structure and pasted them all inside our javascript function' CreateNavigationStreucture()'. Then:

  • We have added document.write(" to the front of each line.
  • We have added "); to the end of each line.
  • We have 'escaped' each double quote character, in the middle of the string, by adding a back slash (\") so that those characters become part of the string rather than being interpretted as terminting a string or beginning a new string.
So this function automates the typing of the HTML code for the navigation structure. So next we need to include 'common.js' in 'index.html' via HTML - that you CAN do, and here is how:

Now let's code that function to toggle the visibility of our popup menu. If it is invisible when we click the 'Products' hyperlink then we want it to show. If it is visible when we click the 'Products' hyperlink then we want it to be hidden. The comments in the code below explain what is going in in the code. You should use comments like this liberally when you are learning javascript.

function DoShowHidePopupMenu(strID)
{
    let divPopup = document.getElementById(strID);

    // Make sure we have a valid ID and that we have a valid popup object
    if (divPopup)
    {
        // If the popup is visible
        if (divPopup.style.display == "block")
            // The hide it
            divPopup.style.display = "none";
        // Else if the popup is hidden
        else if (divPopup.style.display == "none")
            // The show it
            divPopup.style.display = "block";
    }
}
