Introduction

In this section I will teach you to build a basic web page using a variety of common constructs, such as popup menus and navigation structures etc. I will do this in a series of screen capture videos but I will provide minimal ongoing commentary as I do it. Just watch as I edit the code and view the results in the web browser and should should get the idea.

It would be best if you open MS Expression Web and do the same file editing as I am doing in my source file. You can pause the video as you repeat each of the editing steps that I do. By the end of it you should reasonably confident about the coding techniques.

Page structure

To begin with we are going to create the following page structure. It is just one example of how you can structure your web pages and they way you can do so is limited only by your imagination. There is no right or wrong way to structure them.

You will quickly get to know the CSS attributes I am using in the video, as these ones are very frequently used. You can jump back to my CSS page to refresh your memory as to the values available for those CSS attributes. Just follow along in MS Expression Web as I play around with different CSS attributes and values to make the page appear like the in the image above.



In fact why don't you play around with the different values for 'float' and 'display' etc., and see what effect they have on the layout for your page. Try some different values for the 'height' and 'width' atttributes. Try some different back ground colours. This is by far the best way to memorise them and to get to understand what they do. You can do in the following ways:

  • Use comments in you CSS code:

    .navigation
    {
        background-color:green;
        display:inline-block;
        float:/*left;*/right
        height:85%;
        width:15%;
    }

    OR

    .navigation
    {
        background-color:green;
        display:inline-block;
        /*float:left;*/
        float:right
        height:85%;
        width:15%;
    }


  • Just override CSS attributes:

    .navigation
    {
        background-color:green;
        display:inline-block;
        float:left;
        height:85%;
        width:15%;

        float:right;
    }

    In this case the last value ('right') set for 'float' overrides the first value ('left').

Refining the page structure

Company logo and name

In this next video we will add a company logo image and company name to the banner section of the web page. We will get them to sit side by side and align at the vertical center of the banner.



You will have noticed that the 'title' element was not responding to the margin-bottom attribute that I applied to the parent 'banner' element. That is because 'display:inline-block' elements do not quite work the same with margins as do elements using 'display:block'. I therefore had to force it to move up to the vertical center by using the combination of the 'top' and 'position' attributes.

.title
{
    display:inline-block;
    font-size:5em;
    background-color:aqua;
    margin-left:0.1em;
    position:relative;
    top:-22px;
}

By setting 'position' to 'relative', any value you then apply to the 'top' attribute moves the element up or down by that length rather than moving the top edge of the element literally to that position. A negative value means that the element will be moved up by that length while a positive value means that the element will be moved down by that length. This is what I mean when I say you just have to experiment with different combinations of CSS attributes and values until you get the elements to behave as you want them to.

One last thing we will do is to add a page title that matches the document title <title>. Let's remember that this element is not visible on the web page but rather in the web browser tab. You could just manually create a page title for each web page you build, but that is a pain. If you change the title of your web page, or copy it to create a new page, then you would also have to manually change the page title to match. Why bother? Just do this instead at the top of the 'page_content' element.

<script type="text/javascript">
    <document.write("<h1><b>" + document.title + "</b></h1>");
</script>

Or alternatively:

<h1><b><script type="text/javascript">document.write(document.title);</script></b></h1>

Paypal Buttons

There is really no programming skill involved in adding a Paypal button - the Paypal web site generates the HTML code that you then simply paste into your web page. First you will need to create a 'sellers' account and link it to either your bank account or credit/debit card. And then follow this link to the Paypal web site.



We will create a 'buy it now' button.



So here you enter a product name, a price as a decimal number and select the currency from the drop down box. You don't need to enter anything into the product ID field.



Here you can elect tp create a small or large button, you can change the text that appears with the button and you can change the country and language. And there are a bunch of other options you can play around with, but I will leave that to you to figure out over time.





The two options you WILL probably be interested in are the URL edit fields. With these you can specify a URLs on your web site that you can have Paypal to re-direct the user to in the event they make a purchase or cancel their purchase.

Then you simply save the button.
