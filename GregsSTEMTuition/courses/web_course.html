Web programming involves the creation of text based files containing codes and instructions that read by a web browser and used to render the visual elements that make up the web page. It involves 3 distinct types of codes or instructions:

  • HTML tags - acronym for Hypertext Markup Language
        Used to construct the visual and non-visual elements of the page.

  • CSS - acronym for Cascading Style Sheets
        Used to arrange and format the visual elements on the page, and to implement the simple parts of user interaction with the web page.

  • Javascript
        Used to implement the more complex parts of user interaction with the web page.

A web site consists of one or more HTML text files, with the extension .htm or .html, inter-linked together so that the user can switch from one page to another by clicking the hperlinks rather than opening and closing files. A web site will contain one 'home' or default page. Depending on the web server the web site is hosted on the home page is expected to be named any of 'index.htm', index.html', 'default.htm', 'default.html'.

Generally speaking each page in the web site has the same layout and color scheme, and with some form of navigation structure containing hyperlinks to all of the pages in the web siste. But each page will have unique content in one part of the page.
