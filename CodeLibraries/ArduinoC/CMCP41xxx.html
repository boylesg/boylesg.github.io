Purpose:

For Microchip MCP413X, MCP415X, MCP423X, MCP425X. 7/8-Bit Single/Dual SPI Digital POT with Volatile Memory.
5k, 10k, 50k and 100k.

Dependancies:

None.

Defines & types:

#define MAX_DAC_VALUE uint16_t(0b11111111): the maximum 10 bit resistance value that you can send to a channel.

typedef enum{CHANNEL_NONE, CHANNEL_A, CHANNEL_B, CHANNEL_AB} eChannelType: the channel that you want to set a resistance value for – A, B or both.

typedef enum{COMM_SHUTDOWN, COMM_SET} eCommandType: the command you want to send to the chip – shut down or set a resistance value.

typedef enum{RES_5k = 5000, RES_10k = 10000, RES_50k = 50000, RES_100k = 100000} eResistanceType: the total resistance of the chip;

Class CMCP41xxx:

                Constructors & initialisation functions:

CMCP41xxx(const eResistanceType _t eTotalResistance, const uint8_t nSPIChipSelectPin, const uint8_t nResetPin = 255,
            const uint8_t nShutDownPin = 255)

                Sets the total resistance (5k, 10k, 50k or 100k), the SPI chip select pin, the hardware reset pin and the hardware shut down pin.

void begin()

                Must be called to set the correct pin modes etc.

                Functions:

void doSetResistance(const eChannelType eChannel, const uint8_t nValue);

Set a resistance value for the specified channel.
eChannel: the channel (CHANNEL_A, CHANNEL_B or CHANNEL_AB)

nValue: the resistance value from 0 to 255

void doShutdown(const bool bShutDown);

                Do a hardware shut down of the chip.

void doReset();

Do a hardware reset of the chip.

void doShutdownCommand(const eChannelType eChannel);

                Do a command shut down of the chip.

void doZero();

                Set the resistance of all channels to 0.

float doGetR1();

                Return the current resistance of channel 1 or A.

float doGetR2();

                Return the current resistance of channel 2 or B.
