Purpose:

To provide the functionality Arduino class String without using dynamic memory that can cause memory fragmentation of the limited memory available on Ardiuino MCUs.

 

Dependancies:

Arduino assert.h, IPAddress.h & Wire.h. LiquidCrystal_I2C.h for displaying debug output (optional & subject to define in Constants.h).

Constants.h, CBuff.h/cpp, CString.h/cpp, debug.h/cpp, TextFile.h/cpp (optional & subject to a define at the top of debug.h)

 

Constructors:

CString(CBuffBase &buff, const bool bClear = false)

Intialises the CString object with its externally declared CBuff object.

buff: The externall declared CBuff object.

bClear: true if you want the buffer to be initialised to 0's, false if not.

 

CString(CString& strOther)

Intialises the CString object with the CBuff object of another string. So changing the contents of one string also alters the contents of the other.


General Function:

const unsigned int length()

Returns: The length of the internal string up to the first NULL terminator.

 

int capacity()

Returns: the maximum capacity of the internal buffer, beyond any NULL terminator.

 

const unsigned int maxLength()

Returns: the maximum length of any string that can be stored in the internal buffer, taking account of the need for a NULL terminator at the end of the buffer.

 

const bool full()

Returns: true if the length of the string is one character less than the capcity of the buffer.

 

void empty()

Empty the string by changing all characters to NULL terminators.

bool isEmpty()

Returns: true if the length of the string is 0.

nPos: index to start searching from. It defaults to 0 if it is not specified.

Returns: true if the character or substring is found, otherwise false.

Returns: true if the character or substring is found, otherwise false.
