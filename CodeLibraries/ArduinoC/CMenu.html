Purpose:

To build interactive menus for LCD text screens. It requires 4 hardware push buttons or touch sensitive buttons representing ‘increment’, ‘decrement’, ‘enter’ and ‘back’.
There are 4 functions that you call, corresponding to each of those buttons, that scroll the menu and jump to sub-menus etc. It handles scrolling the menu when there are
more items than there are available lines on the LCD screen. The selected menu item is indicated by >… <. Menu items can also contain sub-menus and when your press
‘enter’ the sub-menu is displayed and scrolled. When you press ‘back’ the sub-menu is closed.

 

Dependancies:

Constants.h, Common.h/cpp, CBuff.h/cpp, CString.h/cpp.

 

Class CMenu:

Constructors:

CMenuBase()

                Create an empty menu.

CMenu(const char *cstrName): CMenuBase(cstrName)

CMenu(const __FlashStringHelper *fstrName): CMenuBase(fstrName)

                Creates a menu with the specified name.

                cstrName oe fstrName: the name of the menu.

Functions:

void addMenuItem(const char *cstrName, const uint8_t nID, const CMenuBase *pSubMenu = NULL)

void addMenuItem(const __FlashStringHelper *fstrName, const uint8_t nID, const CMenuBase *pSubMenu = NULL)

                Add an item to the menu.

                cstrName or fstrName: the name of the menu item.

nID: the unique ID of the menu item.

pSubMenu: Pointer to another CMenu object that acts as the sub-menu for this item.

uint8_t EnterPressed()

The function to call when the ‘enter’ push button is pressed. It returns the ID of the menu item. If the menu item contains a sub-menu then the
sub-menu is opened.

void BackPressed()

The function to call when the ‘back’ push button is pressed. If it is a sub-menu open on the LCD screen then it is closed and the parent menu is
displayed.

void IncPressed()

The function to call when the ‘increment’ push button is pressed. Selects the next menu item. If the selected menu item is the last in the list then
it scrolls back to the first item and selects that.

void DecPressed()

The function to call when the ‘decrement’ push button is pressed. Selects the previous menu item. If the selected menu item is the first in the list then
it scrolls back to the last item and selects that.

const char *getMenuItemText()

                Returns the menu item name.

uint8_t getMenuItemNum()

                Returns the number position of the menu item.

uint8_t getMenuItemID()

                Returns the ID of the menu item.

void displayMenu(LiquidCrystal_I2C *pLCD, uint8_t nNumLines, const uint8_t nNumColumns)

                Displays the menu on the specified LCD screen. The name of the menu is displayed on the first line and menu items on subsequentlines.
