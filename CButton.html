Purpose:

To implement the functionality of hardware push, toggle and capacitive touch buttons.

   

 

Dependancies: CBuff.h, CString.h, debug.h

                                Dependancy on CBuff.h and CString.h can be removed by replacing all references to CString with Arduino String.

                                Dependancy on debug.h can be removed by replacing all references to debug with appropriate Serial.print(...) and
                                Serial.println(...) function calls.

               

Functions:

 CPushButton(const uint8_t nPin, const bool bActiveHigh, const uint16_t nMillisTimeOut)

             CToggleButton(const uint8_t nPin, const bool bActiveHigh, const uint16_t nMillisTimeOut)

                                Constructors.

                                nPin: the pin the button is connected to.

                                bActiveHigh: does pressing the button cause a LOW to HIGH or HIGH to LOW transition.

                                nMillisTimeOut: the number of milliseconds until the button will return another clicked event - the click repeat rate.

 

CCapacitiveButton(const uint8_t nSendPin, const uint8_t nReceivePin)

                                nSendPin: the output pin that creates the iscillation.

                                nReceivePin: the input pin that reads the frequency of the oscillation.

 

                void setName(const char *cstrName)

                void setName(const __FlashStringHelper *fstrName)

                                Sets the button's name.

                                cstrName or fstrName: button name.

               

char *getName()

                                Returns: the button's name.

               

void begin()

                                Must be called in the setup function - it puts the pin or pins, associated with the button, in the correct INPUT or OUTPUT mode.

               

bool isClicked()

                                Returns: true if the button is currently clicked, otherwise false.
