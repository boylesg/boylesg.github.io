Purpose:

             To implement a key/value map data structure without using dynamic memory. It is a template class that can use any data types for key and
             value.

 

Dependancies:

                None.

 

Class CMap:

                Constructors:

                                CMap()

                                                Simple constructor with no parameters that creates an empty map.

 

                Functions:

                                bool setAt(KeyType KeyVal, DataType DataVal)

                                                Sets the specified data value at the specified key value. Returns true if there was enough room in the map, otherwise false.

                                                KeyVal: the key value.

                                               DataVal: The data value.

 

                                bool getAt(KeyType KeyVal, DataType &DataVal)

                                                Gets the value at the specified key value. Returns true if the key value was found, otherwise false.

                                                KeyVal: the key value.

                                                DataVal: The data value to be returned.

 

                                bool getNext(uint8_t &nI, KeyType &KeyVal, DataType &DataVal)

                                                Used to iterate through all key/value pairs. Returns true if there was a next key/value pair to get, otherwise false.

                                                nI: the key/value pair number in the arrays.

                                                KeyVal: The key value to be returned.

                                                DataVal: The data value to be returned.

                                uint8_t size()

                                               Returns: the size of the current size of the map.

 

                                uint8_t capacity()

                                               Returns: the size of the capacity of the map.

 

Example Usage:

             #include "Map.h"

CMap<uint8_t, String> mapI2CAddrDevName;

                         void setup()

                         {

                                        mapI2CAddrDevName.setAt(0x0E, String(“LCD”));

                                        mapI2CAddrDevName.setAt(0x27, String(“LCD”));

                                        uint8_t nI = 0, nKey = 0, nVal = 0;

                                        if (mapI2CAddrDevName.getAt(0x0E, nVal))

                                                     Serial.println(nVal);

                                       while (mapI2CAddrDevName.getNext(nI, nKey, nVal))

                                       {

                                                    Serial.print(nI);

                                                    Serial.print(F(", "));

                                                    Serial.print(nKey);

                                                    Serial.print(F(" => "));

                                                    Serial.println(nVal);

                                        }

                          }

 

                          void loop()

                          {

                          }

 

Class CMapPROGMEM:

                Constructors:

                                CMapPROGMEM()

                                                Simple constructor with no parameters that creates an empty map.

 

                                CMapPROGMEM(const KeyType *pKeyArray, const DataType *pDataArray).

                                                Constructor that takes an externally declared PROGMEM array for the key values and another externally declared
                                                PROGMEM array for the stored values.

 

                Functions:

                                bool getAt(const KeyType KeyVal, DataType &DataVal)

Gets the value at the specified key value. Returns true if the key value was found, otherwise false.

KeyVal: the key value.

DataVal: The data value to be returned.

 

                                bool getNext(uint8_t &nI, KeyType &KeyVal, DataType &DataVal)

Used to iterate through all key/value pairs. Returns true if there was a next key/value pair to get, otherwise false.

nI: the key/value pair number in the arrays.

KeyVal: The key value to be returned.

DataVal: The data value to be returned.

 

                                uint8_t size()

                                        Returns: the size of the map.
