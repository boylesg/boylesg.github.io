Unlike public schools, learning is personal project based. There are no lectures, no text books, no work sheets and no tests. You will learn by doing, much like an apprentice electricians and plumbers do.

You will know that you have 'passed' when you can look at a particular project or problem and realise that you know how to implement it or solve it based on what you have learned.

But it will take time and lots of practice, so don't expect to be able to build a web site from scratch or build a wheeled robot within a month. It will more likely be around 12 months before you gain enough knowledge ad confidence to be able to do this sort of thing. And it depends entirely on how much effort you put in outside session times.

Coding and digital electronics is like soccer, football and basketball. Only those with passion for the game and hunger to learn will ever make it to the top leagues. Coding and digital electronics are no different.
